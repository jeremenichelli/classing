Classing

Test Page

This is a page to test the Classing scripts, if your browser supports classList, the script will use it. But, if you the browser doesn't support the API, it includes a fallback for older browsers.

Though you can't see it there's a document element with this Id: test-element and you can check it on your console by the variable name: elem. You can use it to test the following functions:

  • hasClass(elem, "nameOfTheClass")
  • addClass(elem, "nameOfTheClass")
  • removeClass(elem, "nameOfTheClass")
  • toggleClass(elem, "nameOfTheClass")
